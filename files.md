# Project Files

## JavaScript Files

| File | Description |
|------|-------------|
| `index.js` | Main scraper - full workflow: validate company → scrape feed → transform → upsert → fix |
| `company.js` | Validates company via brand search + ANAF + Peviitor APIs (with cache and fallback) |
| `solr.js` | SOLR operations module - exports querySOLR, deleteJobByUrl, upsertJobs + standalone commands |
| `fix-bmw-data.js` | Repair module - fixes company name/CIF for BMW jobs from other scrapers |
| `validate-jobs.js` | Generic job URL validator - checks if URLs are active/expired |
| `src/anaf.js` | ANAF API core module - exports getCompanyFromANAF, searchCompany, getCompanyFromANAFWithFallback |
| `demoanaf.js` | CLI entry point for ANAF module (thin wrapper around src/anaf.js) |
